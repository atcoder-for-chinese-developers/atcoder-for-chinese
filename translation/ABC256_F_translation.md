## 题意

给定一个长为 $n$ 的序列 $a$。$Q$ 次操作。

记 $b_i=\sum^i_{j=1} a_j,c_i=\sum^i_{j=1} b_j,d_i=\sum^i_{j=1} c_j$。

```1 x v```，将 a_x 修改成 v。

```2 x``` ，输出 d_x 对 998244353 取模的结果。

## 数据范围

- $1 \leq n \leq 2 \times 10^5$

- $1 \leq Q \leq 2 \times 10^5$

- $0 \leq a_i \leq 10^9$

- $1 \leq x \leq n$

- $0 \leq v \leq 10^9$

- 所有输入中的数为整数。
