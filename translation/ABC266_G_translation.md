## 题意

已知四个整数 $R,G,B,K$，求有多少种由 ``` R,G,B ``` 组成的字符串满足 ```R``` 出现 $R$ 次，```G``` 出现 $G$ 次，```B``` 出现 $B$ 次。 答案对 $998244353$ 取模。

## 数据范围

- $1 \leq R,G,B \leq 10^6$

- $0 \leq K \leq \min(R,G)$

- 所有数都是整数。

## 输入格式

一行四个整数 ```R G B K```。

## 输出格式

一行一个整数表示答案。

## 样例

### 样例输入1

```
2 1 1 1
```

### 样例输出1

``` 
6
```

### 样例解释1

- ```RRGB```

- ```RGRB```

- ```RGBR```

- ```RBRG```

- ```BRRG```

- ```BRGR```

### 样例输入2

```
1000000 1000000 1000000 1000000
```

### 样例输出2

```
80957240
```
