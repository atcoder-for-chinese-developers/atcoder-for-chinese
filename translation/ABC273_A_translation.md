## 题意

定义函数 $f(x)$ 如下：

- $f(0)=1$；

- $f(k)=k \times f(k-1),k > 0$。

求出 $f(N)$ 的函数值。

## 数据范围

- $0 \leq N \leq 10$。

- 所有输入均为整数。

## 输入格式

第一行输入一个整数 $N$，表示自变量的值。

## 输出格式

第一行输出一个整数，表示函数值。

## 样例

### 样例输入1

```
2
```

### 样例输出1

```
2
```

### 样例解释1

$f(2)=2 \times f(1) = 2 \times 1 \times f(0) = 2 \times 1 \times 1 = 2$

### 样例输入2

```
3
```

### 样例输出2

```
6
```

### 样例解释2

$f(3)=3 \times f(2) = 3 \times 2 = 6$

### 样例输入3

```
0
```

### 样例输出3

```
1
```

### 样例输入4

```
10
```

### 样例输出4

```
3628800
```
