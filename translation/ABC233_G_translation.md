### 题意
有一个长度为 $n\times n$ 的矩阵 $a$，每个元素可能是 `#` 或 `.`。每次操作可以把一个 $d*d$ 的矩阵变成 `.`，代价是 $d$。问你把矩阵全部变成 `.` 的最小花费。
### 数据范围
$2\le n\le50$  
$a_{i,j}=$ `.` 或 `#`

---
### 输入格式
第一行 $1$ 个整数 $n$。  
接下来一个 $n\times n$ 的矩阵表示 $a$
### 输出格式
一行一个整数表示最小花费。

---
### 样例输入1
```
5
##...
.##..
#.#..
.....
....#
```
### 样例输出1
```
4
```
### 样例解释1
左上角花费 $3$ 清空一个 $3\times3$。  
右下角花费 $1$ 清空一个 $1\times1$。

---
### 样例输入2

```
### 样例输出2

```

没有块需要清空。
