## 题意  

给定一个长度为 $N$ 的数组 $A$，问 $A$ 是否是 $1$ 到 $N$ 的排列。

## 数据范围

$1\le N\le 1000$。      
$1\le A_i\le N$。

## 输入格式

第一行，一个整数，$N$。     
第二行，$N$ 个整数，即数组 $A$。
          
## 输出格式

一行 `Yes` 或 `No`。

## 样例

### 输入1 :
```
5
3 1 2 4 5
```

### 输出1 :
```
Yes
```

### 输入2 :
```
6
3 1 4 1 5 2
```

### 输出2 :
```
No
```

### 输入3 :
```
1
1
```

### 输出3 :
```
Yes
```
