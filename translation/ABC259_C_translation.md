## 题意

给定两个串 $S$ 和 $T$ 。记 $S$ 的长度为 $n$ 。

定义一次修改操作为选择一个 $i$ 满足 $1 \leq i < n$ 并且 $s_i=s_{i+1}$。  

询问 $S$ 能否通过若干次修改操作变成 $T$ 。

## 数据范围

$S$ 和 $T$ 都是由小写字母组成的长度在 $2$ 到 $2 \times 10^5$ 之间的字符串。

## 输入格式

两行两个字符串分别为 $S$ 和 $T$ 

## 输出格式

若 $S$ 能通过若干次修改操作变成 $T$ 那么输出 ```Yes ``` 否则输出 ``` No ``` 。

## 样例

### 样例输入1

```
abbaac
abbbbaaac
```

### 样例输出1

```
Yes
```

### 样例解释1

分别在 $i=2,3,6$ 时做修改操作即可。

### 样例输入2

```
xyzz
xyyzz
```

### 样例输出2

```
No
```
