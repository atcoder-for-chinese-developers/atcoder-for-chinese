### 题意 

给定字符串 $s$，求字符 `a` 最后出现的位置。

如果 `a` 没有在 $s$ 中出现，输出 $-1$。

### 数据范围

- $1\le |S| \le 100$
- $s$ 仅由小写字母构成

---

### 输入格式

一行一个字符串 $s$。

### 输出格式

输出答案。

---

### 样例输入1

```
abcdaxayz
```
### 样例输出1

```
7
```

### 样例解释1

`a` 在 $s$ 中出现了 $3$ 次，最后一次出现在位置 $7$。

---

### 样例输入2

```
bcbbbz
```

### 样例输出2

```
-1
```

### 样例解释2

`a` 没有在 $s$ 中出现，输出 $-1$。

---

### 样例输入3

```
aaaaa
```

### 样例输出3

```
5
```
