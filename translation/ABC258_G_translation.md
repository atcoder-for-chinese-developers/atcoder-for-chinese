## 题意

给定一张简单图 $G$，含 $n$ 个顶点。

给定一张邻接矩阵 $a$，若 $a_{i,j}=1$ 则有边，为 $0$ 则无边。

求满足 $1 \leq i < j < k \leq n$ 且 $i$ 和 $j$，$j$ 和 $k$，$i$  和 $k$ 之间右边的三元组 $(i,j,k)$ 的数量。

## 数据范围

- $3 \leq n \leq 3000$

- $a$ 是有向图 $G$ 的一张邻接矩阵。

- 输入的所有数都是整数。

## 输入格式

第一行一个整数 $n$ 表示图的大小。

接下来 $n$ 行每行 $n$ 个数表示边的情况。

## 输出格式

一行一个整数表示答案。

## 样例

### 样例输入1

```
4
0011
0011
1101
1110
```

### 样例输出1

```
2
```

### 样例解释1

$(1,3,4)$ 和 $(2,3,4)$。

### 样例输入2

```
10
0000000000
0000000000
0000000000
0000000000
0000000000
0000000000
0000000000
0000000000
0000000000
0000000000
```

### 样例输出2

```
0
```
