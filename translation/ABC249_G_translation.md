## 题意

有 $n$ 张卡片，第 $i$ 张卡片正面写着数 $a_i$，背面为 $b_i$。

求选择一个集合，满足集合中的卡片正面数的异或和不超过 $k$，求最大的背面数的异或和。

## 数据范围

- $1 \leq n \leq 1000$

- $0 \leq k \leq 2^{30}$

- $0 \leq a_i,b_i \leq 2^{30}(1 \leq i \leq n)$

- 所有输入中的数都是整数。

## 输入格式

第一行两个整数 $n$ 和 $k$。

接下来 $n$ 行每行两个整数 $a_i$ 和 $b_i$。

## 输出格式

一行一个整数表示答案。若找不出一个集合满足异或和不超过 $k$，那么输出 $-1$。

## 样例

### 样例输入1

```
4 2
1 1
3 2
2 2
0 1
```

### 样例输出1

```
3
```

### 样例解释1

选择卡片 $1$ 和卡片 $2$。

### 样例输入2

```
1 2
3 4
```

### 样例输出2

```
-1
```
