## 题意

给定 $5$ 张卡，卡上写的数字分别为 $A,B,C,D,E$，判断其是否满足以下条件：

- 其中三张卡写有相同数字，另两张写有另一个相同数字。

如果满足，输出 `Yes`，否则输出 `No`。

## 样例输入 1
```
1 2 1 2 1
```

## 样例输出 1
```
Yes
```

## 样例解释 1
第 $1,3,5$ 张卡上数字相同，第 $2,4$ 张卡上数字相同。

## 样例输入 2
```
12 12 11 1 2
```

## 样例输出 2
```
No
```

## 数据范围

- $1\le A,B,C,D,E\le 13$
- $A,B,C,D,E$ 均为正数。
- 不存在 $A=B=C=D=E$ 的情况。
