## 题意 

已知一个素数 $P$。

求有多少对整数 $(x,y)$ 满足以下条件：

- $0\le x < P$
- $0\le y < P$
- 存在一个正整数 $n$ 满足 $x^n\equiv y\pmod P$。

答案对 $998244353$ 取模。

## 数据范围

- $2\le P\le 10^{12}$
- $P$ 是素数。

## 输入格式

一行一个数 $P$。

## 输出格式

一行一个数，表示答案。

## 样例

### 样例输入1

```
3
```

### 样例输出1

```
4
```

### 样例解释1

有四对数满足条件，分别是 $(0,0),(1,1),(2,1),(2,2)$。

### 样例输入2

```
11
```

### 样例输出2

```
64
```

### 样例输入3

```
998244353
```

### 样例输出3

```
329133417
```
