### 题意

输入三个数 $a,b,n$，求大于或等于 $n$ 的最小整数 $x$，使 $x$ 同时被 $a$ 和 $b$ 整除。

输出 $x$。

### 数据范围

$1 \leq a,b \leq 100 $，$1 \leq n \leq 20000 $。

---

### 输入格式

```
a
b
n
```

### 输出格式

一行一个数 $x$。

---

### 样例输入 1

```
2
3
8
```

### 样例输出 1

```
12
```

### 样例解释1

$12$ 是大于或等于 $8$ 的最小的整数，可以被 $2$ 和 $3$ 整除。

---

### 样例输入 2

```
2
2
2
```

### 样例输出 2

```
2
```

---

### 样例输入 3

```
12
8
25
```

### 样例输出 3

```
48
```
