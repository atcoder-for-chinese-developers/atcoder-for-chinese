## 题意  

Takahashi 有 $1!,2!,3!,\cdots,10!$ 面值的钱币各 $100$ 个。
问要正好支付 $P$ 元，需要至少几个钱币。

## 数据范围

$1\le P\le 10^7$，
输入的都是整数。     

## 输入格式

第一行，一个整数，$P$。     
          
## 输出格式

一行一个整数，需要至少几个钱币。

## 样例

### 输入1 :
```
9
```

### 输出1 :
```
3
```

### 输入2 :
```
119
```

### 输出2 :
```
10
```

### 输入3 :
```
10000000
```

### 输出3 :
```
24
```

## 样例解释

对于样例一，$1!+2!+3! = 9$，可以证明这是最小的。
