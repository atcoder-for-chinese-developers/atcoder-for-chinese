
## 题意
给出一个正整数 $N$，求出有多少对小于等于 $N$ 的正整数 $(i,j)$ 满足 $i \times j$ 是个完全平方数。

## 数据范围

- $1\le N\le 2\times10^5$
- $N$ 是整数。

## 输入格式

输入是由标准输入提供的，格式如下：

```
N
```

## 输出格式

输出一行答案。

## 样例

### 样例输入1

```
4
```

#### 样例输出1

```
6
```

### 样例解释1

$(1,1),(1,4),(2,2),(3,3),(4,1),(4,4)$ 共六对满足条件。

此外，比如说 $(2,3)$ 就不满足条件，因为 $2\times 3= 6$，且 $6$ 不是一个完全平方数。

### 样例输入2

```
254
```

### 样例输出2

```
896
```
