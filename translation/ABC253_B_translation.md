## 题意
有一个 $H$ 行 $W$ 列的网格图，格子 $(i,j)$ 上有一个字符 $S_{i,j}$，$S_{i,j} =$ `-` 时表示这是一个空格子，$S_{i,j}=$ `o` 时表示这里有一块碎片。

碎片只能上下左右移动一格并且不能移出网格图，请问最小需要移动多少次让碎片移动到同一格？

## 数据范围

- $2 \le H,W \le 100$
- $H,W$ 都是整数。
- $S_{i,j}$ 只有可能是 `o` 或 `-`。
- 只有恰好两个格子上的字符为 `o`。

## 输入格式

输入是由标准输入提供的，格式如下：

```
H W
S1
S2
...
SH
```

## 输出格式

输出一行答案。

## 样例

### 样例一

```
2 3
--o
o--
```

```
3
```

将一行三列的碎片按照 下 左 左 的顺序进行移动，没有比这移动次数更少的移动方法，所以输出 $3$。

### 样例二

```
5 4
-o--
----
----
----
-o--
```

```
4
```
