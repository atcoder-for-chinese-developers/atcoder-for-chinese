### 题意 

给你一个整数 $N$，请你找到最小的符合条件的 $X$。

- $X$ 不小于 $N$。
- 存在一对非负整数 $(a,b)$ 使得 $X=a^3+a^2b+ab^2+b^3$。

### 数据范围

- $N$ 是一个整数。
- $0\leq N\leq 10^{18}$

---

### 输入格式

输入是由标准输入提供的，格式如下：

```
N
```



### 输出格式

直接输出答案。

---

### 样例输入1

```
9
```



### 样例输出1

```
15
```



### 样例解释1

对于任意一个 $X \in [9,14]$，都不存在一对 $(a,b)$ 使其符合条件。

当 $X=15,(a,b)=(2,1)$ 时满足条件。 

---

### 样例输入2

```
0
```



### 样例输出2

```
0
```



### 样例解释2

$N$ 本身满足条件。
